### Main

Главный класс приложения, содержащий точку входа в программу.

Назначение класса:
- реализация консольного меню выбора заданий;
- демонстрация работы каждого задания лабораторной работы;
- запуск и управление выполнением программы.

Класс `Main` не содержит логики решения задач и используется исключительно для демонстрации функциональности разработанных классов.

### Box\<T\>

Обобщённый класс, предназначенный для хранения одного объекта произвольного типа.

Функциональные возможности класса:
- хранение одного объекта типа `T`;
- помещение объекта в коробку;
- извлечение объекта из коробки с последующим обнулением ссылки;
- выбрасывание исключения при попытке поместить объект в непустую коробку;
- проверка состояния коробки на пустоту.

Класс используется для демонстрации работы обобщённых типов.

### ComparableItem\<T\>

Функциональные возможности:
- гарантирует наличие метода `compare`;
- метод принимает объект типа `T`;
- метод возвращает целочисленный результат сравнения.

Интерфейс позволяет изменять тип сравниваемых объектов без изменения самого интерфейса.

### MaxFinder

Класс, предназначенный для поиска максимального числового значения.

Функциональные возможности:
- принимает список объектов класса `Box`, параметризованных числовыми типами;
- извлекает числовые значения из коробок;
- определяет максимальное значение;
- возвращает результат в формате `double`.

Класс использует ограничения обобщённых типов для работы с любыми числовыми значениями.

### FunctionApplier

Класс, реализующий универсальное преобразование элементов списка.

Функциональные возможности:
- принимает список элементов типа `T`;
- принимает объект с методом `apply`;
- применяет переданную функцию к каждому элементу списка;
- возвращает новый список элементов типа `P`.

Используется для преобразования строк, чисел и массивов.

### Filter

Класс, реализующий механизм фильтрации элементов коллекции.

Функциональные возможности:
- принимает список элементов типа `T`;
- принимает объект с методом `test`;
- формирует новый список, содержащий только элементы, удовлетворяющие условию.

Применяется для фильтрации строк, чисел и массивов целых чисел.

### Reducer

Класс, реализующий операцию сведения коллекции к одному значению.

Функциональные возможности:
- принимает список элементов типа `T`;
- принимает функцию объединения двух элементов;
- принимает начальное (identity) значение;
- возвращает итоговое значение типа `T`;
- гарантирует корректную работу с пустыми списками и отсутствие `null`.

Используется для объединения строк, суммирования чисел и подсчёта элементов.

### Collector

Класс, реализующий универсальное накопление элементов в коллекцию.

Функциональные возможности:
- принимает список исходных элементов;
- принимает способ создания результирующей коллекции;
- принимает способ добавления элементов;
- возвращает заполненную коллекцию заданного типа.

Используется для группировки данных, разбиения на подколлекции и получения уникальных значений.
